# Get-VMWareRAHost

## SYNOPSIS
Get details about host by name or list of all hosts from the VMWare Rest API

## SYNTAX

```
Get-VMWareRAHost [-vCenter] <String> [-sessionID] <String> [[-name] <String>] [[-hostID] <String>]
 [[-cluster] <String>] [[-clusterID] <String>] [<CommonParameters>]
```

## DESCRIPTION
Get details about host by name or list of all hosts from the VMWare Rest API
